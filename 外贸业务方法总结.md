# 外贸业务方法总结

## 一、目标市场选择

### 市场优先级（按出单概率）

```
美国 > 越南 > 沙特 > 泰国 > 马来西亚 > 印尼 > 菲律宾 >
尼日利亚 > 南非 > 埃及 > 加拿大 > 墨西哥 > 阿联酋 > 卡塔尔
```

### 市场分层策略

| 市场层级 | 核心特征 | 开发策略 |
|---------|---------|---------|
| **成熟市场**（美/欧/澳） | 门槛高、认证严、利润高 | 重质量、重认证、重规范 |
| **成长市场**（东南亚/中东） | 增长快、需求大、性价比导向 | 快交付、有竞争力、灵活 |
| **新兴市场**（非洲/南美） | 潜力大、单价低、决策快 | 性价比、现货能力、关系 |

### 客户类型定位

```
品牌商/进口商 → 最核心客户（量大、稳定）
工程/项目型客户 → 易出单（项目驱动）
贸易商/批发商 → 量大、利润低（周转快）
零售商/小批发商 → 小单快结（快速变现）
```

## 二、客户开发方法

### 2.1 社媒平台精准搜索

**搜索公式：**
```
(职位关键词) AND (行业/产品词) AND (目标国家)
```

**职位关键词库：**
```
Buyer / Purchasing Manager / Procurement Manager / Sourcing Manager /
Supply Chain Manager / Import Manager / Category Manager /
Project Manager / Technical Manager / EPC Manager /
Owner / Managing Director / CEO / General Manager /
Distribution Manager / Sales Manager / Operations Manager
```

**搜索组合示例：**
```bash
# 搜采购经理（欧美澳）
("Purchasing Manager" OR "Procurement Manager")
AND ("[行业关键词]")
AND ("Australia" OR "UK" OR "Germany" OR "USA")

# 搜项目负责人（中东/非洲）
("Project Manager" OR "EPC Manager" OR "Technical Manager")
AND ("[行业关键词]")
AND ("UAE" OR "Saudi Arabia" OR "Nigeria" OR "Kenya")

# 搜老板层（关键决策人）
("Owner" OR "Managing Director" OR "CEO")
AND ("[行业关键词]")
AND ("目标国家")
```

### 2.2 开发客户四步法

```
第一步：画像素级客户画像（20分钟）
   → 目标市场 + 公司类型 + 决策人职位

第二步：精准搜索 + 筛选（15分钟）
   → 布尔搜索 + 公司规模筛选

第三步：制作定制化资料包（30分钟）
   → 针对具体人/项目，提供价值而非硬推销
   → 解决方案PDF + 产品视频/演示 + 成功案例

第四步：发送请求 + 精准跟进
   → 简洁专业的自我介绍
   → 监控后台数据：阅读/分享/反馈
   → 根据行为数据调整跟进策略
```

### 2.3 开发信核心原则

**高回复率公式：**
```
高回复率 = Basic Search, Target Right + 避开SPAM过滤词 +
           少图片少链接 + 好的标题 + 行文简洁清晰 + 有吸引力的Key Point
```

**必须避开的SPAM高频词：**
```
Free, Discount, Opportunity, Win, Winner, Cheap, Deal,
Debt, Income, Insurance, Loan, Money, Mortgage, Price,
Rate, Profit, Save, Merchant, Specialized, Check, Call
```

**正确开发信结构：**
1. 不提"我从XX平台看到你"——暴露自己是未订阅发送者
2. 不加附件Catalog——容易被垃圾邮件系统拦截
3. 不问"Do you need..."——删除
4. 用专业术语展示价值而不是"We have cheap price"

## 三、客户类型与画像

### 3.1 四大客户类型及跟进策略

| 客户类型 | 特征 | 核心需求 | 跟进策略 |
|---------|------|---------|---------|
| **零售商/小批发商（小B）** | 订单小、下单快、要货急 | 价格、交货期、质量 | 快、准、狠！半个月内成交 |
| **贸易商** | 熟悉中国市场、货比多家 | 价格、性价比、贸易差价 | 找对负责人、预留佣金、配合开拓市场 |
| **OEM/品牌进口商** | 量大、订单稳、注重品牌 | R&D能力、工厂规模、认证、售后 | 展示研发能力、质量标准、认证齐全 |
| **连锁商超/品牌商** | 量大、周期长、决策慢 | 规范、靠谱、配合意识 | 持久战准备，三个月到半年 |

### 3.2 大客户偏爱的供应商特质

```
1. 做事规范 —— 不丢三落四、不拖延马虎
2. 容易沟通 —— 理解对方观点和偏好
3. 考虑周到 —— 多想一步，提前做好
4. 及时响应 —— 迅速回应需求和建议
```

### 3.3 小客户的价值（重要！）

```
1. 容易建立客情 —— 体量小沟通顺畅
2. 了解终端市场 —— 基数大把握需求更精准
3. 未知的发展潜力 —— 客户小≠市场小，可能潜力无限
4. 未知的细分市场 —— 发现新的细分领域
5. 供应商"被需要"程度高 —— 议价时处于有利地位
```

**核心观点：与其撞运气做大客户，不如用心把小客户培养成大客户**

## 四、询盘处理与回复技巧

### 4.1 询盘回复流程

```
分析客户 → 赞许破冰 → 解决需求 → 成功案例展示 → 推动成交
```

**破冰三法：**
- **法一**：告知客户选择的当下最新款/最火款式，先入为主赞美客户眼光
- **法二**：告知公司在客户所在国市场占有率高，提及当地热销款式，展示专业度
- **法三**（针对小客户）：表示愿意100%支持，双赢姿态让其感觉被重视

**推动成交技巧：**
- 做活动/运费折扣
- 本月成交给折扣
- 小起订量支持

### 4.2 模糊询盘处理

**原则：不马上报价，先引导客户互动**

**必问问题：**
1. 订单用途（自用or销售）
2. 客户类型（批发/零售/线上/B2B/平台）
3. 线下联系方式（WhatsApp/Skype/WeChat/Email）

**目标：引起二次回复，建立持续沟通**

### 4.3 价格太贵？三步搞定！

```
Step1：别人买得，你买不得？
   → 发其他客户来访照片、反馈、合作案例链接
   → 告诉客户我们来自很多国家的客户也在这个价格

Step2：将产品拆分介绍，生动形象细节化
   → 展示产品认证/资质
   → 介绍产品细节/卖点
   → 告诉客户质保期限
   → 附产品规格书/参数表

Step3：趁热打铁，正反两面分析
   → 正面：我们的价格是建立在好的质量上，一分价钱一分货
   → 反面：你要的那个价格我们可以给你，但质保可能缩短
```

### 4.4 客户无回复跟进模板

```email
Subject: Kindly check - Did you receive my previous message?

Hi [Name],

Kindly reminding you have you got my last message information?

Once you worked with us you'll have bellow benefit：
1. 1-2pcs Free sample for our new design when you ordered
2. 100% Quality Warranty: quality issue → send new one free in next order
3. Small MOQ SUPPORT: mix style and mix color in small quantity

Best regards,
[Your Name]
```

## 五、客户跟进与分类管理

### 5.1 四类客户及跟进频率

| 客户类型 | 来源 | 跟进频率 |
|---------|------|---------|
| **下单客户** | 平台、官网、开发信、RFQ | 每3天至少一次，发新产品、问候、询问使用情况 |
| **强烈购买意向客户** | 询盘、留言 | 每天跟进一次，保证客户一直记得你 |
| **意向客户** | 平台、官网分类 | 一直保持联系，更新产品、报价、升级信息 |
| **潜在客户** | 开发信 | 发产品推广时不要忘记 |

### 5.2 三种纠结情况的解决方案

| 情况 | 解决方案 |
|------|---------|
| **询价客户发报价后无回复** | 1.邮件不要太长；2.深入挖掘需求信息；3.邮件结束提1-2个客户可能感兴趣的问题 |
| **发PI后客户消失** | 1.从性价比吸引客户；2.时不时跟进状态 |
| **正常交流客户突然"蒸发"** | 属于长期跟进客户，需要耐心，用多种方法开发 |

### 5.3 跟进时机

```
80%客户的成交来自第4次以后的跟进，耐心很重要！
```

## 六、客户接待与谈判

### 6.1 接待前准备清单

```
□ 来访前一周上报部门主管
□ 来往邮件重点及报价表打印
□ 准备好样品（客户感兴趣+可带走）
□ 产品证书、资质文件准备
□ 了解客户国家风俗习惯、饮食习惯、禁忌
□ 接待团队分工：主谈判、副谈判、后勤、拍照
□ 饮料准备：咖啡、茶、可乐、矿泉水
□ 小礼物（如有需要）
□ 正装准备
□ 名片、笔记本、笔、计算器
□ 预定宾馆、餐厅
```

### 6.2 谈判破冰技巧

**换位思考细节：**
- 关心客户行程、住宿、交通是否方便
- 主动提出派车接送机场
- 帮助订展会酒店享折扣

**尊重习俗案例：**
- 接待穆斯林客户：尊重饮食习惯、准备礼拜场所
- 了解各国商务礼仪差异

### 6.3 商务谈判核心

**成交三要素：**
```
1. 舒适的沟通
2. 风险的控制
3. 利润的把控
```

**谈判展示框架：**
```
1. 公司在目标市场的经验积累
2. 产品认证情况（目标国要求的资质）
3. Product lines / Product development / Design / Quality Control
4. 从原材料供应商管理 → 在线生产检验 → 出厂前QC测试 → 入仓
5. 货期保证措施
6. 驻厂人员跟进流程
```

**针对客户顾虑的解决提案：**
- 质量问题 → 建议分批次订单，保证品质一致性
- MOQ问题 → 建议灵活MOQ方案，合作初期降低门槛
- 库存问题 → 建议合理的备货策略或本地化方案

### 6.4 报价策略技巧

**"低价留尾"法：**
```
报个低价，同时规定时效、数量、付款方式、交期等限制性条件
用低价勾起客户兴趣，为将来涨价提供依据
```

**"高价留尾"法：**
```
报高价后，故意规定一些容易达到并有折扣的附加条件
视客户情形选择用哪种
```

**付款方式影响价格：**
```
前TT比例高 → 价格可优惠
付款方式差 → 价格需上浮
账期要求 → 风险成本计入
```

## 七、样品寄送流程

### 寄送前确认
```
□ 客户联系信息（公司名称、地址、电话、邮编、联系人）
□ 样品规格、型号、参数
□ 付款方式（预付/到付/账号）
□ 快递账号有效性
```

### 寄送流程
```
我司付费：准备样品 → 填写快递单 → 代理寄件

客户付费：准备样品 → 填写快递单 → 代理寄件

客户到付账号：确认账号有效 → 准备样品/快递单 →
              准备报关委托书/形式发票(3份) → 如用FedEx需到付保函+营业执照盖章
```

### 后期跟踪
```
1. 询问样品是否顺利到达（表达重视）
2. 引导客户给出具体反馈（满意/不满意）
3. 建立双向稳定沟通联系
```

### 8.1 内容三支柱原则
```
技术内容 40%（产品知识/行业标准/解决方案）
项目内容 30%（实际项目案例/出货故事/客户故事）
```

| 月份 | 主题方向 | 目标区域 | Post类型 |
|------|---------|---------|---------|

| 2月 | 产品选型指南/技术解析 | 欧洲/中东 | 图文+视频 |
| 3月 | 新兴市场项目案例 | 非洲/东南亚 | 图文+客户故事 |

### 8.3 每周量化目标
```
每周Post：2-3条（周二/四/六最佳）
每周评论：至少10个目标客户Post
每周精准搜索+关注：30个潜在客户
每周公司主页Post：1条

月度转化目标：
新增有效关注：100-150人
发送开发消息：20-30封
转化Connection：5-10个
引流到WhatsApp/邮件：3-5个
```

### 8.4 客户开发四步法

```
第一步：精准搜索 + 关注（Follow）
   → 只点关注，不打扰，系统推送动态

第二步：评论互动
   → 专业角度认可型评论，不推销

第三步：发送开发消息（确认值得开发才用）
   → 简洁专业，英文≤300字符
   → 提供价值，而非硬推销

第四步：公司维度开发
   → 找目标公司 → 查看全员 → 找采购/项目负责人
```

## 十一、外贸业务瓶颈期突破

### 成熟外贸人四阶段

```
第一阶段：熟悉产品、流程、学跟单（半年-1年）
第二阶段：外贸流程实践、单证/询盘/谈判（1-2年）
第三阶段：独立成熟业务员、工作3-5年（黄金时代）
第四阶段：瓶颈期——业绩提升遇到挑战
```

### 两条发展路线

| 路线 | 特点 | 适合人群 |
|------|------|---------|
| **M线（管理线）** | 横向发展、带团队、当领导 | 情商高、愿意培养人 |
| **P线（专业线）** | 纵向发展、专注业务、靠专业吃饭 | 喜欢自由、业务能力强 |

### 突破建议

```
管理线：
- 短期内可能损失业务提成
- 从"我"到"我们"的转变
- 知识结构从产品转向管理

专业线：
- 学会找搭档（避免单兵作战）
- 调整心态（收入可能有波动）
- 坚守本行业（不要轻易换行业）
```

## 十二、订单跟单与风险管理

### 12.1 订单跟单流程

```
询盘 → 报价 → 样品确认 → 合同签订 → 生产跟进 → 验货出货 → 收汇 → 售后
```

### 12.2 客户突然要求降价/涨价应对

**客户突然压价：**
1. 了解压价原因（市场竞争？利润压缩？新供应商试探？）
2. 分析己方成本结构，看是否有空间
3. 提供差异化方案（付款方式、交期、批量折扣）
4. 强调价值而非仅谈价格

**客户突然涨价：**
1. 第一时间响应，不要让客户等待
2. 真诚说明原因（原材料/汇率/物流）
3. 提供替代方案或分阶段调整
4. 强调长期合作价值和品质保障

### 12.3 风险控制要点

```
1. 订单确认前：核对所有条款（价格/数量/交期/付款）
2. 生产过程中：定期汇报进度，异常及时预警
3. 出货前：验货报告，文件齐全
4. 收款保障：TT优先，信用证需审核条款
5. 售后跟踪：收货确认，使用反馈
```
